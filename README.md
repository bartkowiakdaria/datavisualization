# 🎨 Data Visualization Projects

This repository contains two projects created as part of the **Data Visualization Techniques** course.  
Both projects were developed using **R** and tools such as **ggplot2**, **shiny**, and **plotly**.  
*All reports and analyses are written in Polish.*

---

## Project 1: Analysis of Student Expenses (Shiny App)

An interactive **Shiny web application** visualizing and analyzing data on **student expenses**.  
The project presents spending habits, category distributions, and comparisons between different students.  
Users can interactively filter data and explore relationships using dynamic plots.

---

## Project 2: The Impact of Remote Work on Mental Health

A data visualization project exploring how **remote work affects mental well-being**.  
The analysis uses **ggplot2** to present correlations between remote work and stress levels.
